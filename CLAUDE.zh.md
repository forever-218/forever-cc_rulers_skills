# cc-forever-rules

Claude Code 行为规则系统，覆盖防卡死、完成验证、简洁性与精确改动、交互质量、执行规则，以及 98+ 种已记录的失败模式。

使用 `/cc-rules` 在当前会话中加载所有规则。

## 快速开始

```bash
# 安装插件
/plugin marketplace add forever-218/cc-rules
/plugin install cc-forever-rules@cc-rules

# 在任何会话中加载规则
/cc-rules
```

## 包含内容

- **防卡死**：防止 30 分钟以上零输出思考、收敛检测、错误重试限制
- **完成验证**：声明完成前审计所有需求，变更后验证
- **简洁与精确改动**：不写推测性代码，不过度工程，只动必须动的
- **交互质量**：执行前问为什么、追溯根因、匹配紧急程度、重建上下文
- **执行规则**：直接执行、完整枚举、澄清歧义、按影响排序优先级
- **详细模式**：38 种交互失败、41 种提前退出、19 种无人值守卡死
